# Excel 处理工具

一个通用友好的 Excel 表格处理工具，支持导入导出和多种表格处理功能。

## 功能特点

- 📤 友好的文件导入导出界面
- 🔄 支持多种 Excel 格式（.xlsx, .xls, .xlsm）
- 🛠️ 可扩展的功能模块框架
- 💻 现代化的 Web UI 界面
- 🧹 数据清洗功能（删除空行、去重、填充缺失值）
- 🔍 数据筛选功能（按条件筛选数据）
- 📊 数据排序功能（多列排序）
- 📈 数据统计功能（缺失值统计、数值统计）

## 安装步骤

1. 安装 Python（建议 3.8 或更高版本）

2. 安装依赖包：
```bash
pip install -r requirements.txt
```

## 使用方法

### 方式一：直接运行 Python 脚本（开发模式）

运行主程序：
```bash
streamlit run main.py
```

程序会自动在浏览器中打开，您可以通过界面：
- 上传 Excel 文件
- 查看表格数据
- 进行数据处理
- 下载处理后的文件

### 方式二：使用打包后的 EXE 文件（推荐，无需安装 Python）

1. **打包成 EXE**（只需执行一次）：
   - 双击运行 `build_exe.bat` 脚本
   - 等待打包完成（约 5-10 分钟）
   - 详细说明请查看 [打包说明.md](打包说明.md)

2. **使用 EXE 文件**：
   - 运行 `dist\Excel处理工具.exe`
   - 程序会自动打开浏览器
   - 无需安装 Python 或任何依赖

**注意**：打包后的 exe 文件可以复制到任何 Windows 电脑使用，无需安装 Python！

## 已实现功能

### ✅ 基础功能
- 文件导入导出
- 数据预览和信息展示

### ✅ 数据清洗
- 删除空行
- 删除重复行（支持指定列）
- 填充缺失值（支持多种填充方式：均值、中位数、众数、前向填充、后向填充、自定义值）

### ✅ 数据筛选
- 按条件筛选（等于、不等于、大于、小于、包含、不包含等）

### ✅ 数据排序
- 多列排序（支持升序/降序）

### ✅ 数据统计
- 缺失值统计
- 数值列统计（均值、中位数、标准差等）

### ✅ 列操作
- 重命名列
- 删除列（单列/多列）
- 添加新列
- 数据类型转换（int, float, str, datetime, bool）

### ✅ 文本处理
- 去除前后空格
- 大小写转换（大写、小写、标题格式、首字母大写）
- 文本替换

### ✅ 数值计算
- 添加计算列（支持 +, -, *, / 运算）
- 分组统计（支持 sum, mean, count, min, max, std）

### ✅ 导出功能
- 导出为 Excel (.xlsx)
- 导出为 CSV (.csv)
- 导出为 JSON (.json)
