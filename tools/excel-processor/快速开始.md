# 快速开始指南

## 🚀 最简单的方式：使用 EXE 文件

### 第一步：打包（只需做一次）

1. 确保已安装 Python 3.8+
2. 双击运行 `build_exe.bat`
3. 等待打包完成（5-10分钟）

### 第二步：使用

1. 打开 `dist` 文件夹
2. 双击 `Excel处理工具.exe`
3. 浏览器会自动打开，开始使用！

**就这么简单！** 之后只需要运行 exe 文件即可，无需再打包。

---

## 💻 开发模式：直接运行 Python

如果您想修改代码或开发新功能：

1. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

2. 运行程序：
   ```bash
   streamlit run main.py
   ```

---

## 📋 使用流程

1. **上传文件** - 在左侧上传 Excel 文件
2. **查看数据** - 在"数据预览"标签页查看数据
3. **处理数据** - 在"处理功能"标签页使用各种功能
4. **下载结果** - 点击"下载文件"保存处理后的数据

---

## ❓ 遇到问题？

- **打包失败？** 查看 [打包说明.md](打包说明.md)
- **exe 无法运行？** 检查是否有杀毒软件拦截
- **其他问题？** 查看控制台窗口的错误信息

---

## 💡 小贴士

- 首次运行 exe 可能需要几秒钟启动
- 如果浏览器没有自动打开，手动访问 http://localhost:8501
- 关闭控制台窗口即可关闭应用
- exe 文件可以复制到任何 Windows 电脑使用
