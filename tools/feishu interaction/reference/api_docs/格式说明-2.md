Business ​

字段​| 类型​| 说明​  
---|---|---  
id​| string​| 业务线 id​  
name​| string​| 业务线名称​  
project​| string​| 所属空间 id​  
labels​| list<string>​| 标签数组​  
role_owners​| Map<string, RoleOwner>​| 默认角色及负责人，Map 的 key 为角色标识（ RoleOwner.role ）​  
watchers​| list<string>​| 默认的关注者数组​  
order​| int64​| 排序字段​  
super_masters​| list<string>​| 业务线超级管理员数组​  
parent​| string​| 父级业务线id​  
disabled​| bool​| 是否弃用​  
level_id​| int​| 层级id，最顶层为1，子节点层级id等于父节点层级id+1​
