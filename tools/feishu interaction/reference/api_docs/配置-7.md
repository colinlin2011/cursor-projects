获取工作项下的流程模板列表

该接口用于获取指定工作项类型下所有“流程模板”列表，对应的平台功能介绍详见​节点流程规则配置或​状态流程配置，对应的权限申请在 权限管理-流程类型 分类下，相关功能介绍详见​权限管理。​

​

​​​

​

请求说明​

  * 请求方式：GET​



  * 请求地址：/open_api/:project_key/template_list/:work_item_type_key​



请求参数​

Header​

接口调用的 Header 配置详见 [Header](../1p8d7djs/4bsmoql6#a8e43b86)。​

Path​

​

参数| 类型| 是否必须| 说明  
---|---|---|---  
project_key| string| 必须| 空间 ID(project_key)，在飞书项目空间双击空间双击空间名称获取。  
work_item_type_key| string| 必须| 工作项类型，系统自动生成，可通过[获取空间下工作项类型](../1p8d7djs/3pjp854w)接口获取。  
  
​

返回参数​

​

参数| 类型| 说明  
---|---|---  
data| list<object>| 遵循[TemplateConf](../1p8d7djs/1x1d372l#9aa45e24)结构规范  
err| object| 请求成功时返回空值，请求失败时返回实际错误信息。  
err_msg| string| 请求成功时返回空值，请求失败时返回实际错误信息。  
err_code| int32| 请求成功时返回0，请求失败返回实际错误码。  
  
​

JSON

复制
