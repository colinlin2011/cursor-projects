服务端 SDK

为了帮助您更轻松地调用 Open API 并降低开发成本，飞书项目提供了 Go 和 Java 语言的服务端 SDK。​

这些 SDK 的主要优势如下：​

1.

将 OpenAPI 的请求和响应参数封装为结构化对象，简化了调用过程。​




2.

自动管理插件访问凭证的生命周期，您无需自行维护。​

​

安装 SDK​

Golang​

执行以下命令安装OpenAPI Golang SDK。​

​

PowerShell

复制

go get -u github.com/larksuite/project-oapi-sdk-golang​

​

Java​

在 settings.xml 文件中添加以下配置：​

​

Java

复制

<settings>​

<profiles>​

<profile>​

<id>lark-project</id>​

<activation>​

<activeByDefault>true</activeByDefault>​

</activation>​

<repositories>​

<repository>​

<id>lark-project</id>​

<url>https://artifacts-cn-beijing.volces.com/repository/meego/</url>​

<releases>​

<enabled>true</enabled>​

</releases>​

<snapshots>​

<enabled>true</enabled>​

</snapshots>​

</repository>​

</repositories>​

</profile>​

</profiles>​

</settings>​

​

在 pom.xml 文件中，添加以下依赖:​

​

Java

复制

<dependency>​

<groupId>com.larksuite.project</groupId>​

<artifactId>oapi-sdk</artifactId>​

<version>{version}</version>​

</dependency>​

Release Notes​

​

SDK 类型​| 版本​| 发布详情​  
---|---|---  
Java SDK​| 1.0.22​| 工作项（Work Item）功能增强：支持按组更新复合字段，并扩展 WBS 视图功能​

  * 新增了 updateCompoundFieldValue 接口，支持按组增加、删除或修改复合字段的值。​



  * WBS 视图的请求现已支持 need_union_deliverable 和 need_schedule_table_agg 查询参数，以及 expand 扩展参数。​



  * 在 WbsViewResponse 响应中增加了关系链实体、路径等字段，以返回更详细的 WBS 视图信息。​
