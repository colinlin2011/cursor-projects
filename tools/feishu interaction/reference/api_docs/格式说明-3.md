字段与属性解析格式

Open API 统一了用户创建和查询的字段格式，以属性和字段的方式进行描述。​

属性：作为工作项的原始信息展示，不以field的方式进行暴露，主要是平台-字段管理页面的系统字段以及内置基础字段。​

字段：区别于原始信息，作为通用的field方式使用，平台-字段管理页面的部分系统字段和用户自定义字段。​

系统字段和用户自定义字段均可通过查询空间字段接口查询字段的配置信息​

富文本格式​

可应用于富文本字段及评论。​

注意！

若在openAPI写入富文本时，直接按照​获取工作项详情的富文本内容入参，所写入的值能够在实例详情页正常展示。​

然而，视图、度量、自动化等场景依赖doc_html、doc、doc_text间满足某种转换规则，如doc_text是对doc_html 中的文字内容提取，上述用法会导致这些场景不可用。​

因此，请在入参时严格参照传参规则输入。​

开放平台侧会在后续提供新能力，在查询时确保各类属性/字段返回结果可直接用于入参。​

​

传参规则​

​

Field​| Value​  
---|---  
type(string)​| paragraph: 文本段落、blank：空白行、checklist：待办列表、ul：无序列表、ol：有序列表、horizontalLine：分割线、table：表格​  
content(array)​| ​  
children.type​| text：文本、hyperlink：超链接、img：图片、linkPreview：链接解析​  
children.text​| 文本内容​  
children.attrs​| backgroundColor、bold、fontColor、fontSize、italic、strikethrough、underline​  
lineAttrs​| indent 缩进数、align、blockquote​  
  
​

富文本分类​

1.

叶子结点（可插入在行内任意位置）：hyperlink、image、linkPreview​




2.

行结构：align、blockquote、indent​




3.

行内容：blank、checklist、horizontalLine、ol、ul、table​




4.

文字属性：backgroundColor、bold、fontColor、fontSize、italic、textDecoration、text​
