# 多维表格 - 获取记录变更历史

**API文档来源**：飞书开放平台  
**最后更新**：2026-01-13

---

## 接口说明

获取多维表格记录的变更历史。注意：飞书多维表格API可能不直接提供变更历史接口，需要通过以下方式获取：

---

## 方法1：通过记录的时间戳筛选

### 获取今日更新的记录
1. 获取所有记录（或使用筛选条件）
2. 检查每条记录的`last_modified_time`
3. 筛选出今日更新的记录

### 实现步骤
1. 计算今日0点和明日0点的时间戳（秒级）
2. 调用获取记录列表API
3. 筛选`last_modified_time`在今日范围内的记录

---

## 方法2：定期快照对比（如果API支持）

1. 定期获取表格快照
2. 对比两次快照的差异
3. 识别新增、修改、删除的记录

---

## 方法3：使用Webhook（如果支持）

1. 配置Webhook监听表格变更
2. 接收变更事件
3. 记录变更历史

---

## 注意事项

1. **API限制**：注意QPS限制
2. **时间戳格式**：使用秒级时间戳
3. **权限要求**：需要读取权限
4. **性能考虑**：大量记录时需要考虑分页和性能
