# 多维表格 - 获取记录列表

**API文档来源**：https://open.feishu.cn/document/server-docs/docs/bitable-v1/app-table-record/list  
**最后更新**：2026-01-13

---

## 接口说明

获取指定多维表格的记录列表，支持筛选、排序、分页等操作。

---

## 请求说明

- **请求方式**：GET
- **请求地址**：`https://open.feishu.cn/open-apis/bitable/v1/apps/{app_token}/tables/{table_id}/records`

---

## 请求参数

### Path参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| app_token | string | 是 | 多维表格的唯一标识符 |
| table_id | string | 是 | 数据表的唯一标识符 |

### Query参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| page_size | int | 否 | 分页大小，默认100，最大500 |
| page_token | string | 否 | 分页标记，首次请求不填，后续请求使用上次返回的page_token |
| filter | string | 否 | 筛选条件，JSON字符串格式 |
| sort | string | 否 | 排序条件，JSON字符串格式 |
| field_names | string | 否 | 指定返回的字段名，多个用逗号分隔 |

### Header参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| Authorization | string | 是 | Bearer {access_token} |
| Content-Type | string | 是 | application/json |

---

## 返回参数

| 参数 | 类型 | 说明 |
|------|------|------|
| code | int | 错误码，0表示成功 |
| msg | string | 错误信息 |
| data | object | 返回数据 |
| data.records | array | 记录列表 |
| data.has_more | boolean | 是否还有更多数据 |
| data.page_token | string | 分页标记 |

---

## 记录字段说明

每条记录包含：
- **record_id**：记录ID
- **fields**：字段数据（JSON对象）
- **created_time**：创建时间（时间戳，秒）
- **last_modified_time**：最后修改时间（时间戳，秒）
- **created_by**：创建人信息
- **last_modified_by**：最后修改人信息

---

## 筛选今日更新记录

### 方法1：使用filter参数
```json
{
  "filter": {
    "conjunction": "and",
    "conditions": [
      {
        "field_name": "最后修改时间",
        "operator": "isGreaterOrEqual",
        "value": [今天0点的时间戳]
      },
      {
        "field_name": "最后修改时间",
        "operator": "isLessThan",
        "value": [明天0点的时间戳]
      }
    ]
  }
}
```

### 方法2：获取所有记录后筛选
1. 获取所有记录
2. 检查每条记录的`last_modified_time`
3. 筛选出今日更新的记录

## 注意事项

1. **时间戳格式**：使用秒级时间戳（不是毫秒）
2. **分页处理**：使用page_token进行分页
3. **字段筛选**：可以使用field_names指定返回的字段
4. **权限要求**：需要`bitable:app:readonly`或`bitable:app`权限
