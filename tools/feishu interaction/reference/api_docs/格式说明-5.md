Open API 错误码

以下是服务端常用错误码列表，具体的错误码结构参考 ​通用说明错误码返回格式一节​

​

Http状态码​| 错误码 err_code​| 错误信息 err_msg(外层)​| 可能的错误原因 & 排查建议​  
---|---|---|---  
403​| 10001​​| No Permission​​| 

  * 没有操作权限​



  * 更新评论：当前操作人不是评论创建人​



  * 更新或删除视图：没有编辑视图的权限​



  * 创建子任务：子任务所在空间和路径空间不匹配​



  * 节点完成/状态流转：没有权限完成​



  * 获取指定的工作项列表：未在对应空间安装插件​



  * 工作项配置修改：操作者没有工作项配置权限​

  
403​| 10002​| Illegal Operation​| 

  * 非法操作​



  * 节点完成/状态流转：当前节点/状态不能流转到指定节点/状态​

  
403​| 10004​| Operation Failed​| 

  * 操作失败​



  * 该操作会导致流程图节点消失，而流程图至少需存在一个节点​

  
401​| 10021​| Token Not Exist​| 

  * 请求头未传plugin_token​

  
401​| 10022​| Check Token Failed​| 

  * plugin_token校验失败​

  
403​| 10210​| code invalid​| 

  * 获取用户访问凭证时，入参的code(授权码)无效​

  
403​| 10211​| Token Info Is Invalid​| 

  * plugin_token信息不合法​



  * 插件token错误，无法解析出具体信息​



  * 如果是插件级token，未传X-USER-KEY​

  
401​| 10301​| Check Token Perm Failed​| 

  * plugin_token权限校验未通过​



  * 没申请当前操作接口的api权限​



  * 申请权限了，但是未发布版本或重新发布版本​



  * 发布版本了，但是未安装或更新插件​



  * 空间ID不存在​



  * 操作者没有对应空间权限​

  
401​| 10302​| Check User Error, User Is Resigned​| 

  * 用户已离职​

  
404​| 13001​| View not exist​| 

  * 视图不存在​

  
403​| 10404​| No Project Permission​| 

  * 当前操作用户没有空间访问权限​

  
429​| 10429​| API Request Frequency Limit​| 

  * 接口请求过于频繁，超过同一token请求同一接口 15qps 限制​

  
429​| 10430​| API Request Idempotent Limit​| 

  * 接口请求幂等限制，可排查header中X-IDEM-UUID幂等串是否冲突​

  
400​| 20001​| Param Request Limit​| 

  * 获取空间详情：传入的空间id超过最大100的限制​

  
400​| 20002​| Page Size Limit​| 

  * 查询评论：page_size超过最大200的限制​

  
400​| 20003​| Wrong WorkItemType Param​| 

  * 获取指定的工作项列表：work_item_type_keys未填​



  * 获取工作项详情/获取指定的工作项列表等接口，目标工作项为状态流，但是设置了"expand"参数:{ "need_workflow":true}​

  
400​| 20004​| Search User Limit​| 

  * 获取指定的工作项列表：user_keys超过最大10的限制​



  * 获取用户详情：最多一次可查询100个用户详情​

  
400​| 20005​| Missing Param​| 

  * 必填的请求参数未填​

  
400​| 20006​| Invalid Param​| 

  * 请求参数不合法，请检查参数与​字段与属性解析格式是否匹配​



  * 如果是创建子任务，提示 target node with role owner config, not support specify 'assignee'"，表明节点负责人和角色绑定，需通过role_assignee字段指定负责人​

  
400​| 20007​| WorkItem Is Already Aborted​| 

  * 工作项已经被终止​

  
400​| 20008​| WorkItem Is Already Restored​| 

  * 工作项已经被恢复​

  
400​| 20009​| Abort Or Restore WorkItem No Reason​| 

  * 终止/恢复工作项，缺失了原因​

  
400​| 20010​| WorkItemType Is Not Same​| 

  * 创建、更新视图：传入的工作项id列表不是同一种工作项类型​

  
400​| 20011​| Input View Is Not Fix View​| 

  * 删除固定视图：目前只支持删除固定视图，想要删除的视图不是固定视图​

  
400​| 20012​| View Is Not In The Input Project​| 

  * 视图的id所属空间不属于参数中的空间​

  
400​| 20013​| Invalid Time Interval​| 

  * 获取指定的工作项列表：时间相关参数，不是毫秒时间戳(13位数字)​

  
400​| 20014​| Project And WorkItem Not Match​| 

  * 工作项所属空间和传入的空间不匹配​

  
400​| 20015​| Field Mix With '-' And Without '-'​| 

  * 存在相同的字段出现在需要返回列表(Without '-') 和 不需要返回列表(With '-')​

  
400​| 20016​| Node Is Not Arrived, Could Not Be operated ​| 

  * 节点未到达，无法进行节点流转​

  
400​| 20017​| Node Is Completed, Could Not Be Completed ​| 

  * 节点已经完成，无法再次完成​

  
400​| 20018​| Node ID Not Exist In Workflow​| 

  * 节点不存在当前工作项的节点流配置中​



  * 解决方案： 请检查工作项配置中是否存在该节点id​

  
400​| 20019​| Invite Bot Limit 5​| 

  * 拉取群的机器人数量超过5个​

  
400​| 20020​| Bot App_ids Empty​| 

  * 未填拉取群的机器人​

  
400​| 20021​| ChatID Not Belong WorkItem​| 

  * 群id不属于参数中的工作项​

  
400​| 20024​| Uploaded File Size Limit 100M​| 

  * 上传文件大小限制100M​

  
400​| 20025​| Field_key/Field_alias Missing​| 

  * 字段的key和对接标识都缺失​

  
400​| 20026​| FlowType Is Error, Please Convert To Status FlowType​| 

  * 查询的工作项是状态流工作项​

  
400​| 20027​| FlowType Is Error, Please Convert To Node FlowType​| 

  * 查询的工作项是节点流工作项​

  
400​| 20028​| Workitem Ids Limit 50​| 

  * 传入的工作项id数量限制50​

  
400​| 20029​| Unsupported Field Type​| 

  * 不支持更新的字段类型(字段key会在msg中返回)​

  
400​| 20032​| DifferentSchedule Set Owner Invalid​| 

  * 差异化排期未指定用户更新排期，或者指定的用户超过一个​

  
400​| 20033​| Update Field Invalid​| 

  * 不能更新该字段​

  
400​| 20037​| Node Is not completed, Could Not Be Rollback ​| 

  * 节点未完成，无法回滚​

  
400​| 20038​| Required Field Is Not Set​| 

  * 节点完成/状态流转时，必填字段未填写​

  
400​| 20039​| Plugin Token Must Have User Key, But X-USER-KEY Is Not Set In Request Header​| 

  * 使用的是应用token，请求头中必须带上X-USER-KEY​

  
400​| 20040​| Request Form Is Null​| 

  * 上传附件时，传入的表单是空(content-type未选择multipart/form-data)​

  
400​| 20041​| Field RoleOwner Must Be Set In FieldValuePairs​| 

  * 创建工作项必须传入role_owners字段​

  
400​| 20042​| X-User-Key Is Wrong, Please Check First​| 

  * 未填X-User-Key或者传入的user_key错误​

  
400​| 20043​| View Ids Limit 10​| 

  * 查询视图列表时，最多只能传入10个视图id​

  
400​| 20044​| WorkItem Has Been Disabled​| 

  * 工作项已被禁用，无法查询到元数据​

  
400​| 20045​| Comment And WorkItem Not Match​| 

  * 评论不属于指定的工作项​

  
400​| 20046​| Task ID Not Exist In Workflow​| 

  * 工作流中不存在该子任务​

  
400​| 20047​| Role_Assignee and Assignee Can't Be Set Together​| 

  * 更新或创建子任务，role_assignee和assignee不能同时传入​

  
400​| 20048​| Role_Assignee's Role Is Not Match​| 

  * 更新或创建子任务时，传入的role_assignee里面的role与节点绑定的role不匹配​

  
400​| 20049​| TenantGroupID Is Wrong​| 

  * 仅渠道用户使用，填入的TenantGroupId错误​

  
400​| 20050​| 「field_key」Field Option Value Is Wrong​| 

  * 更新或创建工作项时，传入的字段选项值错误​

  
400​| 20051​| FieldLinkedStory Value Is Wrong​| 

  * 填入的field_linked_story字段值错误​

  
400​| 20052​| IssueOperator Value Is Wrong​| 

  * 缺陷的operator角色负责人填写错误，可对比获取工作项详情接口返回值的role_owners字段​

  
400​| 20053​| IssueReporter Value Is Wrong​| 

  * 缺陷的reporter角色负责人填写错误，可对比获取工作项详情接口返回值的role_owners字段​

  
400​| 20055​| Search Result Bigger Than 2000, Please Check Your Search Params​| 

  * 查询结果超过2000个，请重新设置筛选条件​

  
400​| 20056​| Only WorkFlow Mode Can Be Aborted Or Restored​| 

  * 只有工作流模式可以终止和恢复​

  
400​| 20057​| Search ProjectKeys And SimpleNames Limit 10​| 

  * 搜索时，传入的ProjectKeys和SimpleNames的并集不能超过10个​

  
400​| 20058​| SearchUser.Role And SearchUser.FieldKey Can't appear together​| 

  * 搜索时，在SearchUser这个结构中，Role和FieldKey不能同时出现​

  
400​| 20059​| SearchUser.FieldKey Or SearchUser.Role Can't Appear Alone，SearchUser.UserKeys Must Appear​| 

  * 搜索时，在SearchUser这个结构中，UserKeys如果为空，Role或FieldKey不能单独传入​

  
400​| 20060​| WorkItemTypeKey Or RelationWorkItemTypeKey Is Not Match In Field Configuration​| 

  * 工作项类型或关联工作项类型，与关联关系的配置不匹配​

  
400​| 20061​| RelationKey Type Is Not Relation In Configuration​| 

  * 指定关联关系的字段类型，不是关联类型​

  
400​| 20062​| RelationType Error​| 

  * 指定的RelationType不存在，目前只支持0：字段key,1：字段alias​

  
400​| 20063​| Search Operator Error,​| 

  * 搜索的操作错误，不同的参数可使用的操作符不同​

  
400​| 20064​| Search Option Size Too Large​| 

  * 搜索指定的选项个数超过限制，最多可传入50个选项值​

  
400​| 20065​| Search Param Key Not Support StateFlow : Param Key is ​| 

  * 当前参数不支持状态流筛选​

  
400​| 20066​| Search Signal Only Support len=1 When Operator Is 「=」 or 「!=」​| 

  * 搜索系统外信号，当操作符是=或者!=时，数组长度只能是1​

  
400​| 20067​| Search Signal Not Support Value :​| 

  * 搜素系统外信号，传入的值不支持筛选​

  
400​| 20068​| Search Param Is Not Support, ​| 

  * 指定的参数不支持筛选​

  
400​| 20069​| Search Param Value Error, ​| 

  * 搜索传入的参数值异常​

  
400​| 20070​| Field is InValid​| 

  * 上传附件时指定的字段已失效​

  
400​| 20071​| Search People Not Support Issue​| 

  * 搜索指定参数是people时，不支持缺陷工作项​

  
400​| 20072​| Conjunction Value Only Support 「AND」、「OR」​| 

  * 搜索的Conjunction仅支持且、或​

  
400​| 20080​| Query length must be larger than 0,smaller than 200​| 

  * 综搜查询中query必填同时长度限制小于200​

  
400​| 20081​| Query type is not supported​| 

  * 综搜查询中目前仅支持查询工作项和视图​

  
400​| 20082​| Action is not supported​| 

  * 子任务状态更新接口中只有回滚和确认操作​

  
400​| 20083​| Duplication Field Exist, FieldKey : ​| 

  * 创建工作项时，传入的字段重复​

  
400​| 20090​| Your request has been intercepted.​| 

  * 请求或操作被插件拦截了​

  
404​| 30005​| WorkItem Not Found​| 

  * 工作项未找到，可能是工作项已删除、查询的工作项id不正确、查询的工作项类型和工作项id不匹配​

  
404​| 30006​| User Not Found​| 

  * header中传入的user_key未找到对应用户​



  * 获取用户详情接口中：​



  * 未查到指定用户​



  * 使用了虚拟token，只能查到插件协作者相关信息​



  * 确认插件是否共享到插件市场，共享到插件市场的为 ISV 插件，不支持调用「获取用户详情」API​

  
404​| 30007​| Workflow Not Found​| 

  * 工作项中未找到节点流​

  
404​| 30008​| Business Not Found​| 

  * 空间下的业务线未找到​

  
404​| 30009​| Field Not Found​| 

  * 字段未在字段配置中，无法更新或创建​

  
404​​| 30010​| Stateflow Not Found​| 

  * 工作项中未找到状态流​

  
404​| 30011​| Node Not Found In Workflow​| 

  * 节点在工作项的节点流配置中未找到​

  
404​| 30012​​​| State Not Found In Stateflow​| 

  * 状态在工作项的状态流配置中未找到​

  
404​| 30015​| Record Not Found​| 

  * 更新评论失败，评论ID不存在，需检查参数评论ID​

  
400​| 9999​| Invalid Param​| 

  * 数据结构不匹配，检查入参​

  
400​| 1000051942​| commercial usage exceeded​| 

  * 商业化用量超限​

  
500​| 50006​| RPC Call Error This type does not support​| 

  * 未知错误，请联系技术支持​

  
500​| 50006​| openapi system err, please try again later​| 

  * 服务瞬时超载，可尝试重试恢复​
