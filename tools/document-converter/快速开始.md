# 快速开始指南

## 第一步：安装依赖

打开终端，进入工具目录：

```bash
cd "p:\Cursor Project\tools\document-converter"
pip install -r requirements.txt
```

## 第二步：运行工具

```bash
streamlit run main.py
```

工具会自动在浏览器中打开（通常是 http://localhost:8501）

## 第三步：转换文档

1. **上传文档**：点击"选择要转换的文档"，选择你的PDF/Word/Excel文件
2. **设置输出**：可以选择输出目录，或使用默认位置
3. **开始转换**：点击"🔄 开始转换"按钮
4. **下载结果**：转换完成后，可以预览和下载Markdown文件

## 使用示例

### 转换标准PDF

1. 上传 `01. [新评估基准]GB 智能网联汽车 组合驾驶辅助系统安全要求_工信部官方公开征求意见版本_2509.pdf`
2. 设置输出目录为 `p:\Cursor Project\study-systems\autonomous-safety-study\materials\01_Standards\`
3. 点击转换
4. 生成 `01. [新评估基准]GB 智能网联汽车 组合驾驶辅助系统安全要求_工信部官方公开征求意见版本_2509.md`

### 转换Word文档

1. 上传从飞书Wiki导出的Word文档
2. 设置输出目录为 `p:\Cursor Project\study-systems\autonomous-safety-study\materials\02_FeishuWiki\`
3. 点击转换
4. 生成对应的Markdown文件

## 提示

- 转换后的Markdown文件可以直接被AI助手读取
- 建议将转换后的文件保存到学习系统的materials目录
- 大文件转换可能需要一些时间，请耐心等待
