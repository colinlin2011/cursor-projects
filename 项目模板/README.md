# 新工具项目模板

这是一个新工具项目的模板，您可以基于此创建新的工具。

## 📁 项目结构

```
工具名称/
├── main.py              # 主程序入口
├── config.py            # 配置文件
├── requirements.txt     # 依赖包列表
├── README.md           # 项目说明文档
├── utils/              # 工具模块目录（可选）
│   └── __init__.py
└── build_exe.bat       # 打包脚本（如果需要打包）
```

## 🚀 快速开始

1. 复制此模板目录
2. 重命名为您的工具名称
3. 修改 `main.py` 开始开发
4. 更新 `requirements.txt` 添加依赖
5. 编写 `README.md` 说明文档

## 📝 注意事项

- 保持项目独立性
- 每个工具有自己的依赖管理
- 建议添加详细的 README 说明
