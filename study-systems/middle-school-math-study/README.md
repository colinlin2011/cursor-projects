# 初中数学学习系统

## 项目介绍

这是一个专注于初中数学的系统化学习项目。通过AI导师（Cursor AI助手）的苏格拉底式教学法，帮助你全面掌握符合中国教学大纲的初中数学知识，培养数学思维，提高数学素养。

## 学习目标

- 全面掌握初中数学知识体系（数与代数、图形与几何、统计与概率）
- 培养逻辑思维、抽象思维和数学思维能力
- 提高数学问题分析和解决能力
- 建立扎实的数学基础，为高中学习做好准备
- 培养对数学的兴趣和信心

## 知识体系

### 数与代数
- **有理数**：正负数、数轴、绝对值、有理数运算
- **整式**：单项式、多项式、整式运算、乘法公式
- **方程**：一元一次方程、二元一次方程组、一元二次方程
- **函数**：一次函数、反比例函数、二次函数

### 图形与几何
- **平面几何**：点线面、角、平行垂直、三角形、四边形、圆
- **立体几何**：立体图形认识、表面积和体积
- **坐标几何**：平面直角坐标系、函数图像
- **图形变换**：平移、旋转、轴对称、相似

### 统计与概率
- **统计**：数据收集整理、平均数、中位数、众数、统计图
- **概率**：随机事件、概率计算、频率估计概率

### 综合与实践
- 数学建模
- 数学探究活动
- 实际应用问题

## 使用方法

### 1. 开始学习
在Cursor中打开本项目，AI导师会根据`CURSOR.md`中的指令自动进入教学模式。

### 2. 学习流程
1. **初始评估**：AI导师会先了解你的年级、知识背景和学习目标
2. **制定计划**：根据你的情况制定个性化学习计划
3. **互动学习**：采用苏格拉底式提问法，通过问答进行学习
4. **进度记录**：每次学习后，AI导师会帮你更新`progress/study-tracker.md`
5. **会话记录**：学习内容会自动记录到`sessions/`目录

### 3. 查看进度
定期查看`progress/study-tracker.md`了解：
- 各知识模块的掌握程度
- 已学知识点和待学内容
- 错题记录和复习计划

### 4. 复习巩固
根据会话记录和进度跟踪，定期复习已学内容，确保知识掌握牢固。

## 学习建议

### 1. 循序渐进
按照知识的内在逻辑，从基础到高级，逐步深入。不要跳跃式学习。

### 2. 理解为主
注重理解数学概念和方法，而不仅仅是记住公式。理解后才能灵活应用。

### 3. 多练习
数学需要多练习才能熟练掌握。在学习新知识后，及时做相关练习题。

### 4. 联系实际
将数学知识与实际生活联系起来，提高学习兴趣和理解深度。

### 5. 及时复习
定期复习已学内容，防止遗忘。建议采用间隔重复的方法。

### 6. 记录错题
建立错题本，记录做错的题目和原因，定期复习错题。

## 学习路径

### 七年级（初一）
**重点内容**：
- 有理数的运算
- 整式的运算
- 一元一次方程
- 几何图形初步
- 相交线与平行线

**学习建议**：打好基础，熟练掌握基本运算和基本概念。

### 八年级（初二）
**重点内容**：
- 全等三角形
- 轴对称
- 整式的乘法与因式分解
- 分式
- 一次函数
- 平行四边形

**学习建议**：几何证明和函数概念是重点，需要多练习和思考。

### 九年级（初三）
**重点内容**：
- 一元二次方程
- 二次函数
- 圆
- 相似
- 锐角三角函数
- 概率初步

**学习建议**：综合运用所学知识，提高解题能力，为中考做准备。

## 注意事项

1. **系统性学习**：建议按照年级和章节顺序学习，不要跳跃式学习
2. **理解为主**：注重理解概念和方法，而不仅仅是记住公式
3. **多练习**：数学需要多练习，及时做相关练习题
4. **记录总结**：及时记录学习内容和错题，便于复习和总结
5. **保持耐心**：数学学习需要时间和耐心，不要急于求成

## 开始学习

### 快速开始步骤

1. **在Cursor中打开本项目目录**
   - 选择 `File` → `Open Folder`
   - 导航到 `study-systems/middle-school-math-study`

2. **开始学习对话**
   - 在Cursor聊天窗口输入："我想开始学习初中数学"
   - 或："我是[年级]学生，想学习[具体主题]"

3. **AI导师会**：
   - 先了解你的年级和知识水平
   - 制定个性化学习计划
   - 采用苏格拉底式提问法开始教学
