# 材料库说明

## 目录结构

```
materials/
├── 01_Standards/             # 存放标准PDF文件
│   └── 组合辅助驾驶强标.pdf
│
├── 02_FeishuWiki/            # 从飞书Wiki导出的文档
│   └── [按需添加导出的文档]
│
├── 03_Polarion/              # 从Polarion导出的文档
│   └── [按需添加导出的文档]
│
└── 04_工作文档/              # 协作编写的工作文档
    ├── 条款对应矩阵.md        # 标准条款与证据文档的对应关系
    ├── 材料索引清单.md        # 所有材料的索引
    ├── 符合性检查清单.md      # 符合性检查清单
    └── 整改措施清单.md        # 整改措施清单
```

## 使用说明

1. **标准文档**：将标准PDF文件放在 `01_Standards/` 目录
2. **飞书导出**：从飞书Wiki导出的文档放在 `02_FeishuWiki/` 目录
3. **Polarion导出**：从Polarion导出的文档放在 `03_Polarion/` 目录
4. **工作文档**：所有协作编写和管理的文档放在 `04_工作文档/` 目录

## 注意事项

- PDF文件可以直接放在对应目录
- 如果导出为Markdown格式，也可以直接使用
- 工作文档使用Markdown格式，方便AI助手直接编辑
