# 自动驾驶功能安全学习系统

## 项目介绍

这是一个专注于自动驾驶功能安全领域的系统化学习项目。通过AI导师（Cursor AI助手）的苏格拉底式教学法，帮助你深入掌握ISO 26262功能安全标准、ISO 21448 (SOTIF) 预期功能安全、组合辅助驾驶强标（中国标准）以及L3/L4自动驾驶技术。

## 学习目标

- 全面掌握ISO 26262功能安全标准的核心概念和实践方法
- 深入理解SOTIF (ISO 21448) 预期功能安全的理论和应用
- 精通组合辅助驾驶强标（中国标准）的要求和合规性评估
- 掌握L3/L4高级别自动驾驶系统的安全设计、开发和验证
- 能够独立进行ASIL评估、安全分析和验证确认工作

## 知识领域

### ISO 26262 功能安全标准
国际汽车功能安全标准，涵盖从概念到生产的整个安全生命周期，包括：
- 功能安全概念和ASIL等级评估
- 系统、硬件、软件层面的安全开发
- 安全分析和验证确认方法

### ISO 21448 (SOTIF) 预期功能安全
针对自动驾驶系统的预期功能安全标准，重点关注：
- 已知和未知不安全场景的识别
- 功能改进和验证方法
- 与ISO 26262的协同应用

### 组合辅助驾驶强标（中国标准）
中国针对组合辅助驾驶系统的强制性标准，包括：
- 功能安全要求
- 预期功能安全要求
- 网络安全要求
- 合规性评估和实施方法

### L3/L4 自动驾驶技术
高级别自动驾驶系统的核心技术，包括：
- 系统架构和安全设计
- 冗余与降级策略
- 人机交互安全设计
- 测试验证方法

## 使用方法

### 1. 开始学习
在Cursor中打开本项目，AI导师会根据`CURSOR.md`中的指令自动进入教学模式。

### 2. 学习流程
1. **初始评估**：AI导师会先了解你的知识背景和学习目标
2. **制定计划**：根据你的情况制定个性化学习计划
3. **互动学习**：采用苏格拉底式提问法，通过问答进行学习
4. **进度记录**：每次学习后，AI导师会帮你更新`progress/study-tracker.md`
5. **会话记录**：学习内容会自动记录到`sessions/`目录

### 3. 查看进度
定期查看`progress/study-tracker.md`了解：
- 各知识领域的掌握程度
- 已学知识点和待学内容
- 知识缺口和学习建议

### 4. 复习巩固
根据会话记录和进度跟踪，定期复习已学内容，确保知识掌握牢固。

## 学习路径建议

### 初级阶段（1-2个月）
1. ISO 26262基础概念和术语
2. ASIL评估方法
3. 安全生命周期概述
4. 基础安全分析技术

### 中级阶段（2-3个月）
1. 系统层面安全开发
2. 硬件和软件安全开发
3. SOTIF基础概念
4. 场景识别和分类方法

### 高级阶段（3-4个月）
1. L3/L4系统架构设计
2. 组合辅助驾驶强标深入理解
3. 复杂安全分析和验证
4. 实际工程案例研究

### 专家阶段（持续学习）
1. 前沿技术跟踪
2. 标准更新和解读
3. 实际项目应用
4. 经验总结和分享
