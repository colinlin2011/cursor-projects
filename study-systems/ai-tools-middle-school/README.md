# 中学生AI工具学习系统

## 项目介绍

这是一个专注于中学生AI工具使用的系统化学习项目。通过AI导师（Cursor AI助手）的项目式学习和探究式学习，帮助中学生深入理解AI工具，培养数字素养和批判性思维，提高学习效率，为未来学习和工作做好准备。

## 学习目标

- 深入理解AI的基本原理和应用
- 掌握核心AI工具的高级使用技巧
- 培养批判性思维和独立思考能力
- 提高学习效率和学习质量
- 培养数字素养和创新能力
- 学会合理使用AI，避免过度依赖

## 知识体系

### AI基础理论
- AI的基本原理
- AI工具的分类
- AI的局限性和挑战

### 核心AI工具深度使用
- **文本生成AI**：高级提示词工程、应用场景、批判性思维
- **图像生成AI**：提示词技巧、应用场景、版权和伦理
- **代码生成AI**：基础使用、应用场景、最佳实践
- **多模态AI工具**：综合应用

### 学习应用
- 学习辅助（作业辅导、知识学习、论文写作）

### 安全与伦理
- 网络安全
- AI使用伦理
- 批判性思维

### 进阶应用
- AI工具的组合使用
- 创新应用
- 未来准备

## 使用方法

### 1. 开始学习
在Cursor中打开本项目，AI导师会根据`CURSOR.md`中的指令自动进入教学模式。

### 2. 学习流程
1. **初始评估**：AI导师会先了解你的年级、学习目标和对AI的了解程度
2. **制定计划**：根据你的情况制定个性化学习计划
3. **互动学习**：采用项目式和探究式学习，通过实际项目和案例进行学习
4. **进度记录**：每次学习后，AI导师会帮你更新`progress/study-tracker.md`
5. **会话记录**：学习内容会自动记录到`sessions/`目录

### 3. 查看进度
定期查看`progress/study-tracker.md`了解：
- 各知识模块的掌握程度
- 已学知识点和待学内容
- 项目作品记录

### 4. 复习巩固
根据会话记录和进度跟踪，定期复习已学内容，确保知识掌握牢固。

## 学习建议

### 1. 深度理解
不仅学习如何使用，更要理解为什么这样用，培养批判性思维。

### 2. 实践为主
通过实际项目和案例来学习，多尝试，多探索。

### 3. 批判性思维
对AI的输出保持批判性思考，验证准确性，识别偏见和错误。

### 4. 合理使用
避免过度依赖AI，保持独立思考能力，结合AI和人类智慧。

### 5. 安全第一
注意网络安全和隐私保护，遵守学术诚信和版权规范。

### 6. 持续学习
AI技术发展迅速，保持学习，跟上最新发展。

## 开始学习

### 快速开始步骤

1. **在Cursor中打开本项目目录**
   - 选择 `File` → `Open Folder`
   - 导航到 `study-systems/ai-tools-middle-school`

2. **开始学习对话**
   - 在Cursor聊天窗口输入："我想开始学习AI工具"
   - 或："我是[年级]学生，想学习[具体主题]"

3. **AI导师会**：
   - 先了解你的年级和学习目标
   - 制定个性化学习计划
   - 采用项目式和探究式学习开始教学
