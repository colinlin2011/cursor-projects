## 贡献者分类

### 按贡献类型分类

- **能力贡献者**：贡献AI能力（Skill、Agent、工具、MCP）
- **知识贡献者**：贡献知识、模式、经验教训
- **配置贡献者**：贡献共享配置
- **文档贡献者**：贡献文档和模板

### 按贡献频率分类

- **活跃贡献者**：最近3个月有贡献
- **定期贡献者**：定期贡献（每月至少1次）
- **一次性贡献者**：只贡献过1次

## 如何成为贡献者

### 贡献能力

1. 开发能力并验证可用性
2. 使用贡献脚本提交：
   ```bash
   python scripts/contribute_capability.py --capability-path [路径] --name [名称]
   ```
3. 填写贡献信息
4. 自动记录到贡献者列表

### 贡献知识

1. 识别可共享的知识
2. 使用知识模板编写
3. 提交到团队知识库
4. 记录到贡献者列表
