## 注册表说明

本注册表用于管理团队共享的所有AI能力，包括Skill、Agent、MCP和本地工具。所有能力都需要在此注册后才能被团队使用。

### 能力状态说明
- **可用**：能力已注册且可用
- **不可用**：能力已注册但当前不可用
- **维护中**：能力正在维护，暂时不可用
- **已废弃**：能力已废弃，不再使用

### 使用统计说明
- **使用次数**：能力被使用的总次数
- **使用人数**：使用过该能力的团队成员数量
- **平均评分**：使用者对该能力的平均评分（1-5分）
- **最后使用**：最后一次使用的时间

## 能力分类

### 按功能分类

#### 飞书集成类
- 飞书API封装、文档协作、表格操作、多维表格操作等

#### 文档处理类
- Markdown转换、文档生成、模板处理等

#### 数据分析类
- 数据提取、统计分析、报告生成等

#### 工作流类
- 自动化流程、任务管理、进度跟踪等

### 按类型分类

- **Skill能力**：单一功能的AI能力
- **Agent能力**：具有自主决策能力的AI代理
- **本地工具**：封装为Skill的本地工具
- **MCP能力**：通过MCP协议集成的外部服务

### 按场景推荐

## 贡献能力

### 贡献流程

1. 开发能力并验证可用性
2. 使用贡献脚本提交：
   ```bash
   python scripts/contribute_capability.py --capability-path [路径] --name [名称]
   ```
3. 填写贡献信息
4. 自动注册到本注册表
