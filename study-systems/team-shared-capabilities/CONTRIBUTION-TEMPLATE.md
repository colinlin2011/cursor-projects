## 贡献流程

### 步骤1：准备能力
1. 开发能力并验证可用性
2. 编写完整文档（使用本模板）
3. 准备示例代码
4. 测试所有功能

### 步骤2：使用贡献脚本
```bash
python scripts/contribute_capability.py \
    --capability-path capabilities/skills/my_capability \
    --name "我的能力" \
    --description "能力描述" \
    --category "工具类" \
    --contributor "姓名"
```

### 步骤3：填写信息
脚本会引导你填写：
- 能力基本信息
- 功能描述和使用场景
- 依赖和配置要求
- 使用示例

### 步骤4：自动注册
脚本会自动：
- 检查能力完整性
- 生成贡献文档
- 更新团队注册表
- 记录贡献者信息

### 步骤5：团队审核（可选）
- 团队审核能力质量
- 提供改进建议
- 确认注册

---

## 注意事项

1. **完整性**：确保能力功能完整、文档齐全、示例可用
2. **通用性**：确保能力具有通用性，可以被多个场景使用
3. **文档化**：提供清晰的文档和使用示例
4. **测试**：确保能力经过充分测试
5. **维护**：承诺维护能力，及时响应问题
