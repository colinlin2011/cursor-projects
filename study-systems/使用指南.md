# 学习系统使用指南

## 快速开始

### 第一步：选择学习系统

在 `study-systems/` 目录下有两个学习系统：

1. **autonomous-safety-study** - 自动驾驶功能安全学习系统
2. **middle-school-math-study** - 初中数学学习系统

### 第二步：在Cursor中打开项目

1. 打开Cursor编辑器
2. 选择 `File` → `Open Folder`（或使用快捷键）
3. 导航到你要学习的系统目录，例如：
   - `p:\Cursor Project\study-systems\autonomous-safety-study`
   - 或 `p:\Cursor Project\study-systems\middle-school-math-study`
4. 点击"选择文件夹"

### 第三步：开始学习对话

在Cursor的聊天窗口中，输入以下任一方式开始：

**方式1：直接开始学习**
```
我想开始学习自动驾驶功能安全
```
或
```
我想开始学习初中数学
```

**方式2：查看学习系统信息**
```
请先介绍一下这个学习系统，然后帮我制定学习计划
```

**方式3：从特定主题开始**
```
我想学习ISO 26262的基础概念
```
或
```
我想学习有理数的运算
```

## 详细使用流程

### 1. 首次使用 - 初始评估

当你第一次使用学习系统时，AI导师会：

1. **了解你的背景**
   - 询问你的知识水平
   - 了解你的学习目标
   - 确定你的学习起点

2. **制定学习计划**
   - 根据你的情况制定个性化计划
   - 推荐学习路径
   - 设定学习目标

**示例对话：**
```
你：我想开始学习自动驾驶功能安全

AI导师：你好！很高兴成为你的学习导师。在开始之前，我想了解一下：
1. 你目前对ISO 26262功能安全标准有多少了解？
2. 你的学习目标是什么？（比如：工作需要、考试准备、兴趣学习等）
3. 你希望重点学习哪个方面？（ISO 26262、SOTIF、L3/L4、组合辅助驾驶强标）
```

### 2. 日常学习流程

#### 开始一次学习会话

1. **打开Cursor**，确保在正确的学习系统目录中
2. **在聊天窗口输入**：
   ```
   我想继续学习[主题名称]
   ```
   或
   ```
   我想学习[具体知识点]
   ```

3. **AI导师会**：
   - 先询问你已经了解的内容（苏格拉底式提问）
   - 根据你的回答补充讲解
   - 通过提问验证你的理解
   - 提供实际案例和应用

#### 学习过程中的互动

**如果你理解了：**
```
我理解了，请继续下一个知识点
```

**如果你有疑问：**
```
我对[具体概念]还不太理解，能再详细解释一下吗？
```

**如果你想看例子：**
```
能给我一个实际应用的例子吗？
```

**如果你想练习：**
```
我想做一些练习题来巩固这个知识点
```

### 3. 学习后的记录

每次学习结束后，AI导师会：

1. **自动创建会话记录**
   - 在 `sessions/` 目录下创建新的会话文件
   - 文件名格式：`SESSION-YYYYMMDD-001.md`

2. **更新学习进度**
   - 更新 `progress/study-tracker.md`
   - 标记已掌握的知识点
   - 记录学习笔记

**你可以说：**
```
请帮我记录这次学习的内容
```

### 4. 查看学习进度

#### 查看整体进度

1. 打开 `progress/study-tracker.md` 文件
2. 查看：
   - 各知识点的掌握程度（未开始/学习中/已掌握）
   - 学习笔记
   - 知识缺口
   - 学习计划建议

#### 查看历史会话

1. 打开 `sessions/` 目录
2. 查看之前的会话记录文件
3. 复习之前学过的内容

**你可以说：**
```
请帮我查看一下我的学习进度
```
或
```
我想复习一下之前学过的[主题]
```

### 5. 复习和巩固

#### 主动复习

```
我想复习一下[主题名称]
```

AI导师会：
- 回顾之前学过的内容
- 通过提问检查你的掌握程度
- 补充讲解薄弱环节

#### 查看错题（数学系统）

在数学学习系统中，你可以：

```
我想看看我的错题记录
```

或

```
我想复习一下做错的题目
```

## 实用技巧

### 技巧1：利用文件上下文

Cursor可以读取项目中的文件，所以你可以：

1. **打开相关文件**（如 `CURSOR.md`、`study-tracker.md`）
2. **在聊天中引用**：
   ```
   根据我的学习进度，我应该学习什么？
   ```
   AI会读取 `study-tracker.md` 来了解你的进度

### 技巧2：指定学习方式

```
我想用苏格拉底式提问法学习[主题]
```

```
我想先看一个实际案例，再学习理论
```

### 技巧3：调整学习节奏

```
这个知识点讲得太快了，请慢一点
```

```
我已经掌握了基础，可以讲得更深入一些
```

### 技巧4：跨主题学习

```
我想了解ISO 26262和SOTIF的关系
```

```
我想学习如何将有理数运算应用到实际问题中
```

## 常见问题

### Q1: 如何切换学习系统？

**A:** 在Cursor中打开另一个学习系统的目录即可。两个系统是独立的，可以同时使用。

### Q2: 学习记录会丢失吗？

**A:** 不会。所有学习记录都保存在项目文件中：
- 会话记录：`sessions/` 目录
- 学习进度：`progress/study-tracker.md`

### Q3: 如何从上次中断的地方继续？

**A:** 
1. 打开 `progress/study-tracker.md` 查看上次的学习进度
2. 对AI说："我想继续上次的学习"
3. AI会读取进度文件，从上次的地方继续

### Q4: 可以同时学习多个主题吗？

**A:** 可以，但建议按顺序学习，因为很多知识点是递进的。如果确实需要，可以明确告诉AI：
```
我想同时学习[主题A]和[主题B]
```

### Q5: 如何让AI记住我的学习偏好？

**A:** 在对话中明确说明你的偏好，AI会记录在会话文件中。例如：
```
我喜欢通过实际案例来理解概念
```

## 学习建议

### 1. 保持连续性
- 建议每天或定期学习
- 每次学习后让AI记录进度
- 定期复习已学内容

### 2. 主动提问
- 不要只是被动接受
- 主动提出疑问
- 要求实际案例和应用

### 3. 理论结合实践
- 学习理论后要求练习题
- 尝试将知识应用到实际问题
- 记录实际应用中的疑问

### 4. 定期回顾
- 每周查看一次学习进度
- 复习之前的会话记录
- 巩固薄弱环节

**"我想开始学习！"**
