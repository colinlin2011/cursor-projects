如果需要重新生成图片，可以：

1. **使用mermaid-cli**（推荐）：
   ```bash
   npm install -g @mermaid-js/mermaid-cli
   mmdc -i input.mmd -o output.png -w 1920 -H 1080
   ```

   - 访问 https://mermaid.live/
   - 复制Mermaid代码
   - 导出为PNG

3. **运行转换脚本**：
   ```bash
   python convert_mermaid_to_images.py
   ```
