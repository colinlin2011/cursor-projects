### 处理建议

1. **使用在线工具**（最简单）：
   - 访问 https://mermaid.live/
   - 复制Mermaid代码
   - 导出为PNG图片
   - 插入Word文档

2. **使用Mermaid CLI**（适合批量处理）：
   ```bash
   npm install -g @mermaid-js/mermaid-cli
   mmdc -i input.mmd -o output.png
   ```
