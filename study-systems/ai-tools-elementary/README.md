# 小学生AI工具学习系统

## 项目介绍

这是一个专注于小学生AI工具使用的系统化学习项目。通过AI导师（Cursor AI助手）的游戏化教学和项目式学习，帮助小学生安全、有效地使用深度学习AI工具，培养数字素养，激发创造力和学习兴趣。

## 学习目标

- 了解AI的基本概念和应用
- 掌握适合小学生使用的AI工具
- 培养安全使用AI的意识
- 通过AI工具激发创造力和学习兴趣
- 培养数字素养和批判性思维
- 学会合理使用AI，避免过度依赖

## 知识体系

### AI基础认知
- 什么是AI（用简单易懂的语言）
- AI在我们生活中的应用
- AI能做什么，不能做什么
- AI和人类的关系

### 基础AI工具使用
- **文本生成AI**：提问回答、故事创作、作业辅导
- **图像生成AI**：根据描述生成图片、创意设计
- **学习辅助AI**：作业辅导、知识点解释、练习题生成

### 创意项目实践
- 故事创作项目
- 绘画创作项目
- 学习辅助项目

### 安全与伦理
- 网络安全和隐私保护
- AI使用伦理
- 时间管理和合理使用

### 进阶应用
- 综合项目实践
- 创新思维培养

## 使用方法

### 1. 开始学习
在Cursor中打开本项目，AI导师会根据`CURSOR.md`中的指令自动进入教学模式。

### 2. 学习流程
1. **初始评估**：AI导师会先了解你的年级、兴趣和对AI的了解程度
2. **制定计划**：根据你的情况制定个性化学习计划
3. **互动学习**：采用游戏化和项目式学习，通过实际操作进行学习
4. **进度记录**：每次学习后，AI导师会帮你更新`progress/study-tracker.md`
5. **会话记录**：学习内容会自动记录到`sessions/`目录

### 3. 查看进度
定期查看`progress/study-tracker.md`了解：
- 各知识模块的掌握程度
- 已学知识点和待学内容
- 项目作品记录

### 4. 复习巩固
根据会话记录和进度跟踪，定期复习已学内容，确保知识掌握牢固。

## 学习建议

### 1. 安全第一
始终注意网络安全和隐私保护，不分享个人信息。

### 2. 合理使用
合理使用AI工具，避免过度依赖，保持学习和娱乐的平衡。

### 3. 多实践
通过实际操作和项目来学习，多尝试，多探索。

### 4. 培养思考
在使用AI的同时，培养独立思考能力，不盲目相信AI的回答。

### 5. 家长参与
鼓励家长了解学习内容，确保安全使用。

### 6. 分享作品
与同学、老师、家长分享自己的作品，获得反馈。

## 开始学习

### 快速开始步骤

1. **在Cursor中打开本项目目录**
   - 选择 `File` → `Open Folder`
   - 导航到 `study-systems/ai-tools-elementary`

2. **开始学习对话**
   - 在Cursor聊天窗口输入："我想开始学习AI工具"
   - 或："我是[年级]学生，想学习[具体主题]"

3. **AI导师会**：
   - 先了解你的年级和兴趣
   - 制定个性化学习计划
   - 采用游戏化和项目式学习开始教学
